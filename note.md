# Wimpy Books upgrade summary

Implemented:
- login-gated access to reading and uploads
- preview-first book discovery on the home, store, and display pages
- uploaders can choose free or paid access and set prices
- promoted ad placements for sponsored books
- comments and ratings on each book page
- a darker, glassy, animated UI with a light/dark toggle
- a lightweight Node.js backend with auth, book, comment, rating, and purchase routes

Run:
- npm start
- npm test
